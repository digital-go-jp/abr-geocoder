東京都千代田区紀尾井町1番3号

// 都道府県を省略しても変換されるはず
千代田区紀尾井町1-3

// 大字（紀尾井町）からでも変換されるはず
紀尾井町1-3

// 漢数字や全角数字でも問題なく変換されるはず
東京都千代田区紀尾井町一の三

東京都千代田区紀尾井町壱番地三号

東京都千代田区紀尾井町壱丿３  // 「丿」全角カタカナ
東京都千代田区紀尾井町壱ﾉ３   // 「ﾉ」半角カタカナ
東京都千代田区紀尾井町壱之３

// 3番地とデジタルビル９９９の間にスペースが入ることを期待
東京都千代田区紀尾井町1丁目3番地デジタルビル９９９

// 1-3と東京ガーデンテラスの間にスペースが入ることを期待
